How to install and use this design system

Quick Start

This design system is a plain CSS/HTML system with zero dependencies. To use it in your project:

Include the Tokens

<link rel="stylesheet" href="css/design-tokens.css" />

This gives you all CSS custom properties — colours, type scale, spacing, shadows, and radii.

Include Components

<link rel="stylesheet" href="css/components.css" />

Add this for pre-built component styles (the dithering button, etc).

Import Google Font

The system uses Inter. Add this to your <head>:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
